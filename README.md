# Pokémon Top Trumps Game

A Python-based Top Trumps style game using Pokémon data from PokéAPI.

## Features
- Random Pokémon selection
- Turn-based stat comparison
- Draw replay logic
- Score tracking
- Table-based final results

## How to Run
1. Install Python
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python main.py

## Technologies
- Python
- PokéAPI
