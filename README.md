# Pokémon Top Trumps Game

A Python-based Top Trumps style game using Pokémon data from PokéAPI.
##  Game Description
This game randomly assigns Pokémon to two players.  
On each turn, a player selects a stat (height or weight) from their Pokémon.  
The Pokémon stats are compared and the player with the higher value wins the round.

The game supports multiple rounds, draw replays, turn-based stat selection, and displays final scores in a table.


## Features
- Random Pokémon selection
- Turn-based stat comparison
- Draw replay logic
- Score tracking
- Table-based final results

## How to Run
1. Install Python
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python main.py

## Technologies
- Python
- PokéAPI
